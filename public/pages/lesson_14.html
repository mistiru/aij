    大山おおやまさん の 誕生日たんじょうび

    ヤン 「もしもし、 大山おおやまさん です か。 ヤン です。」

    大山おおやま 「 アメリカ に いる ヤンさん ？ お元気げんき です か。」

    ヤン 「はい。 元気げんき です。 大山おおやまさん 、 お誕生日たんじょうび、 おめでとうございます。」

    大山おおやま 「ああ、 ヤンさん 、 私わたし の 誕生日たんじょうび を まだ 覚おぼえていました か。 ありがとうございます。」

    ヤン 「もちろん です。 でも 今年ことし は 一緒いっしょ に 誕生日たんじょうびパーティー が できませんでした ね。 もう パーティー を しました か。」

    大山おおやま 「ええ。 昨日きのう 会社かいしゃ の 友達ともだち と ケーキ を 食たべたり、 ダンス を したり して 楽たのしかった です よ。 明日あした は 家族かぞく と レストラン へ 行いきます。」

    ヤン 「（ア）」

    大山おおやま 「来月らいげつ 仕事しごと で アメリカ へ 行いきます から ヤンさん にも 一度ど 会あいたい です ね。」

    ヤン 「本当ほんとう です か。 （イ） その とき は 電話でんわ を ください。」

    問題もんだい

    （ア） には 何なに を 入いれます か。

    • そう でした
    • そう です か
    • そう しましょう
    • そう します

    （イ） には 何なに を 入いれます か。

    • じゃあ
    • 多分たぶん
    • どうも
    • あれから

    ヤンさん は どうして 大山おおやまさん に 電話でんわ を しました か。

    • 大山おおやまさん に 会あいたい から 電話でんわしました。
    • 大山おおやまさん が アメリカ に 行いく から 電話でんわしました。
    • 大山おおやまさん が パーティー を する から 電話でんわしました。
    • 大山おおやまさん の 誕生日たんじょうび から 電話でんわしました。

    正ただしい もの は どれ です か。

    • 大山おおやまさん は 明日あした 会社かいしゃ の 友達ともだち と パーティー を します。
    • 大山おおやまさん は 昨日きのう 会社かいしゃ の 友達ともだち と パーティー を しました。
    • 大山おおやまさん は 明日あした 誕生日たんじょうび から 友達ともだち と レストラン へ 行いきます。
    • 大山おおやまさん は 昨日きのう 誕生日たんじょうび だった から 家族かぞく と パーティー を しました。
