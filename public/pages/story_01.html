おむすびころりん

むかしむかし おじいさん と おばあさん が いました。 ある日、 おじいさん は、 おばあさん に でっかい おむすび を 作つくってもらい、 山 へ 木 を 切きり に 出かけました。

とんかんとんかん 木 を 切っている うち に お昼ひる に なりました。 「さあて、 お腹なか が 空すいた。 飯めし に でも する か。」 おじいさん は 木 の 切り 株かぶ に 座すわって、 おむすび の 包つつみ を 開ひらきました。 その 途端とたん、 おむすび が ころん と 落おっこち、 ころころ 転ころがりました。 「こら まて、 おむすび どこ へ 行く。」

おじいさん は 慌あわてて おむすび を 追おいかけました。 おむすび は ころころ ころころ 転がって、 ころりん すっとん 穴あな の 中 に 落ちました。 「こりゃ 困こまった ぞ。」 おじいさん は 何とも 中 を 除のぞきました。

すると、 かわいい 鼠ねずみ が 出て来て、 「おじいさん、 おむすび ありがとう。」 と 言いました。 「なんて かわいい 鼠 じゃ。」 おじいさん は 悔くやしい の も 忘わすれて 鼠 を 見つめました。 「お礼れい に おらたち の 国くに へ 案内あんないします。」 「でも、 こんな 小さな 穴 に は 入はいらない。」 「大丈夫だいじょうぶ。 おら の 尻尾しっぽ に つかまって、 目 を つぶってください。」

おじいさん は、 鼠 の 尻尾 を つかんで 目 を つむりました。 すると なんだか いい 気持きもち に なって、 ふわふわ 進すすんでいきました。 「はあい、 目 を 開あけて。」 鼠 が いいました。 はっとして 目 を 開けたら、 なんと 御殿ごてん のような 家いえ が あって、 広ひろい 台所だいどころ で 鼠たち が おもち を 突ついていました。
